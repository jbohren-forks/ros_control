\subsection{\-Detailed \-Description}
\-A basic pid class. 

\-This class implements a generic structure that can be used to create a wide range of pid controllers. \-It can function independently or be subclassed to provide more specific controls based on a particular control loop.

\-In particular, this class implements the standard pid equation\-:

$command = -p_{term} - i_{term} - d_{term} $

where\-: \par
 
\begin{DoxyItemize}
\item $ p_{term} = p_{gain} * p_{error} $ 
\item $ i_{term} = i_{term} + \int{i_{gain} * p_{error} * dt} $ 
\item $ d_{term} = d_{gain} * d_{error} $ 
\item $ d_{error} = (p_{error} - p_{error last}) / dt $ 
\end{DoxyItemize}

given\-:\par
 
\begin{DoxyItemize}
\item $ p_{error} = p_{state} - p_{target} $.

\subsection{\-Usage}\label{classcontrol__toolbox_1_1Pid_Usage}
\-To use the \doxyref{\-Pid}{p.}{classcontrol__toolbox_1_1Pid} class, you should first call some version of \doxyref{init()}{p.}{classcontrol__toolbox_1_1Pid_a7ce13102646c5c4970e8c49493a9333f} (in non-\/realtime) and then call \doxyref{update\-Pid()}{p.}{classcontrol__toolbox_1_1Pid_a3b67ef1db5948f1da331e54171c2667e} at every update step. \-For example\-:

\begin{DoxyVerb}

double position_desi_ = 0.5;
...
ros::Time last_time = ros::Time::now();
while (true) {
  ros::Time time = ros::Time::now();
  double effort = pid.updatePid(currentPosition() - position_desi_, time - last_time);
  last_time = time;
}
