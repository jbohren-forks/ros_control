\subsection{\-Detailed \-Description}
\-Sets up services for quickly changing the gains for a \doxyref{control\-\_\-toolbox\-::\-Pid}{p.}{classcontrol__toolbox_1_1Pid} \doxyref{\-Pid}{p.}{classcontrol__toolbox_1_1Pid} object. 

\-The \doxyref{\-Pid\-Gains\-Setter}{p.}{classcontrol__toolbox_1_1PidGainsSetter} class provides services for changing the gains of \doxyref{\-Pid}{p.}{classcontrol__toolbox_1_1Pid} objects over \-R\-O\-S. \-It advertise the \char`\"{}set\-\_\-gains\char`\"{} service in the \-Node\-Handle's namespace, with a type of control\-\_\-toolbox/\-Set\-Pid\-Gains.

\-To use the object, add pids to the gains setter and then call \doxyref{advertise()}{p.}{classcontrol__toolbox_1_1PidGainsSetter_abf3f47ec6d3afca0ff691a7ba0066e7d}. \-The \doxyref{\-Pid\-Gains\-Setter}{p.}{classcontrol__toolbox_1_1PidGainsSetter} will then update the gains of all the \doxyref{\-Pid}{p.}{classcontrol__toolbox_1_1Pid} objects when you call the set\-\_\-gains service. (\-If you wish to have \-Pids with different gains, then you should use multiple \doxyref{\-Pid\-Gains\-Setter}{p.}{classcontrol__toolbox_1_1PidGainsSetter} objects).
